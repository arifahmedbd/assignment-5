Question-1: What are the different ways to select an element in the DOM?

We can select an element in 6 different ways.

  • By ID- document.getElementById
  • By class name- document.getElementsByClassName
  • By Tag Name- document.getElementsByTagName
  • By Attribute Name- document.getElementsByName
  • By Querry Selector- document.querySelector
  • By Querry Selector All- document.querySelectorAll

Question-4: What is event bubbling in the DOM?

Event bubbling is a system where an event clicked for a child element it bubble up to it's parent element.
